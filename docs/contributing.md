# Contributing

Thank you for your interest in contributing to PDF Merger!

Please read the full [CONTRIBUTING.md](https://github.com/ucoruh/pdf-merger/blob/main/CONTRIBUTING.md) on GitHub for detailed guidelines on:

- Reporting bugs
- Suggesting features
- Submitting pull requests
- Coding standards
- Development setup

## Quick Start

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes and add tests
4. Ensure all tests pass: `dotnet test`
5. Commit and push to your fork
6. Open a pull request

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://github.com/ucoruh/pdf-merger/blob/main/CODE_OF_CONDUCT.md).
